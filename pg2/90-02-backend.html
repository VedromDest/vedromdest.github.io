Herhalingsoefening Backend (Api + EF Core)

Doel

Ontwerp en implementeer een beknopt systeem voor het beheer van treinabonnementen via een RESTful api.

Het Centraal Station in Antwerpen door FrancisVK

Deliverables

  • MySQL DDL/DML script

  • Werkende dotnet solution

  • Postman collection die je gebruikt hebt om te testen

Requirements

Business

  • Een klant

    • heeft een naam, voornaam en email adres

      • het email adres is uniek

    • kan abonnementen nemen

  • Een abonnement is geldig

    • tussen twee stations

    • voor een bepaalde periode (datum van - datum tot)

    • voor een bepaalde klasse (1 of 2)

  • Een station heeft

    • een naam

    • (g)een verwarmde wachtruimte

De geldigheidsperiode voor abonnementen van dezelfde klant tussen dezelfde twee stations, mag niet overlappen!

Domein Model

Technisch

  • Gebruik MySQL

  • Gebruik Entity Framework Core scaffolding

  • Gebruik een drielagen backend architectuur

    • Api (gebruik aparte request en response contracten)

    • Service

    • Storage

Api

Klanten
POST /api/klanten -> 1 maken
GET /api/klanten -> lijst van alle
GET /api/klanten/{id} -> 1 specifieke
Abonnementen
POST /api/abonnementen -> 1 maken
GET /api/abonnementen -> lijst van alle
GET /api/abonnementen/{id} -> 1 specifieke
Stations
POST /api/stations -> 1 maken
GET /api/stations -> lijst van alle
GET /api/stations/{id} -> 1 specifieke
PUT /api/stations/{id} -> update verwarmde wachtruimte ja/nee
    (dit is een geldige PATCH kandidaat)
