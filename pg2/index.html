Welkom bij Programmeren Gevorderd 2!

Alle noemenswaardige broncode uit de lessen is online beschikbaar.

Doel

In dit vak worden enkele belangrijke technieken aangeleerd die essentieel zijn in het werkveld. Er ligt een grote focus op het verhogen van efficiëntie en productiviteit.

  • RESTful HTTP Api’s

  • Meerlagige Backends

  • Entity Framework Core

  • Refactoring

  • Dapper Micro ORM

  • Asynchroon & Parallel Programmeren

  • Document Database

Studiemateriaal

Alle studiemateriaal wordt online beschikbaar gesteld.

  • Code zal na elke les beschikbaar gesteld worden

  • Deze online cursus is altijd integraal beschikbaar.

Het wordt sterk aangemoedigd de online documentatie van Microsoft te raadplegen om zich te verdiepen.

Aanpak

Lessen worden gegeven als een mix van theorie en live coding tutorial. Alle code komt integraal online.

Je wordt niet verondersteld mee te volgen op je eigen laptop.

Er worden optionele oefeningen beschikbaar gesteld voor studenten die materie willen inoefenen.

Evaluatie

Consulteer de Studiewijzer voor meer informatie betreffende de evaluatie.

Broncode In Deze Cursus

Tijdens de lessen wordt er aan live coding gedaan. Dit heeft als gevolg dat de code uit de les niet exact overeen zal komen met deze cursus.

Laat je hierdoor niet in de war brengen. Je zal snel ondervinden dat deze afwijkingen zeer oppervlakkig zijn en het geheel conceptueel identiek is aan de cursus. Immers: de lessen volgen de cursus, niet omgekeerd.
