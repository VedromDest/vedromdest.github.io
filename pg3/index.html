Welkom bij Programmeren Gevorderd 3!

Doel

  • Iteratief bouwen van een data-driven async .Net webapi systeem.

  • Azure DevOps development flow leren kennen.

  • Systeem installeren in de Azure cloud.

Studiemateriaal

  • Code zal na elke les beschikbaar gesteld worden in het Azure DevOps repository.

  • Lesopnames worden beschikbaar gesteld - ook voor de dagopleiding.

  • Deze online cursus is altijd integraal beschikbaar.

Het wordt sterk aangemoedigd de online documentatie van Microsoft te raadplegen.

Aanpak

De rode draad doorheen de lessen, te beginnen met les twee, is de ontwikkeling van een systeem voor een fictief bedrijf dat pakjes levert, zoals PostNL of UPS.

De verstandige student zal aan de hand van de lessen, lesopnames en deze cursus het project in parallel bouwen om zo de geziene stof in te oefenen.

Gestaag zullen er nieuwe business en technische requirements aangebracht worden. De implementatie zal besproken en getoond worden tijdens de les. Het eindresultaat is een cloud applicatie met beknopte functionaliteit.

Evaluatie

De student kiest een onderwerp waarrond deze individueel een cloud project zal bouwen en deployen. Geen pakjesbedrijf of architecturale copy-paste van het project uit de lessen, maar iets grofweg equivalent qua complexiteit.

Het doel is werk op te leveren dat goed genoeg is om in een bedrijf gebruikt te worden.

Consulteer de Studiewijzer voor meer informatie betreffende de evaluatie.

Tooling

Microsoft investeert sinds 2014 enorm in een platformonafhankelijk .Net ecosysteem dat werkt op Windows, macOS en Linux.

We gebruiken onderstaande cloud tooling

  • Azure DevOps

  • Azure Portal

Indien je jouw Azure For Students Benefits in het verleden al activeerde (via de Azure Portal), kan het zijn dat deze rechten verlopen zijn. Zolang je student bent, kan je deze benefits verlengen.
We gebruiken onderstaande desktop tooling

  • .Net 8 LTS SDK

  • Azure CLI

  • Azure Data Studio

  • Azure Storage Explorer

  • Docker Desktop

  • Git

  • Postman

  • Visual Studio Code met "C# Dev Kit" extension

Waarom geen klassieke Visual Studio?

Deze moderne tools verschijnen steeds vaker in het werkveld. Ze leren gebruiken is belangrijk om de kansen van de student op de arbeidsmarkt te verhogen.

Broncode In Deze Cursus

Tijdens de lessen wordt er aan live coding gedaan. Dit heeft als gevolg dat de code uit de les, en dus in het Git repository, niet exact overeen zal komen met deze cursus.

Laat je hierdoor niet in de war brengen. Je zal snel ondervinden dat deze afwijkingen zeer oppervlakkig zijn en het geheel conceptueel identiek is aan de cursus. Immers: de lessen volgen de cursus, niet omgekeerd.
